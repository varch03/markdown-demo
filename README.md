# ✨ Welcome to GitHub Markdown

## 🧠 What Is Markdown?
Markdown is a lightweight markup language used to format text.

## ✍️ Basic Syntax
- **Bold**
- _Italic_
- `Inline code`
- [Link](https://github.com)

## 📁 Where Is Markdown Used?
- READMEs
- GitHub Pages
- Wikis
- Issues & PRs
